# Fluxo - Sistema de Logística Premium

Fluxo es una plataforma integral para la gestión de inventarios, pedidos y auditorías, diseñada con un enfoque en la experiencia de usuario y la eficiencia operativa.

## Características Principales

*   **📦 Inventario en Tiempo Real:** Control total de productos, marcas y categorías.
*   **🚚 Gestión de Pedidos:** Flujo completo desde la creación hasta la recepción.
*   **📋 Auditoría Inteligente:**
    *   Recepción de mercancía con validación inmediata.
    *   **Auditoría Física:** Conciliación de inventario y ajustes automáticos.
    *   Modo Histórico con detalles de lectura.
*   **✨ Interfaz Premium:**
    *   Modo Oscuro (Glassmorphism).
    *   Modo Claro (Minimalista/Apple-style).
    *   Animaciones fluidas y respuestas táctiles.

## Tecnologías

*   **Frontend:** React `19`, TypeScript, Vite.
*   **Estilos:** TailwindCSS, Lucide Icons.
*   **Datos:** Arquitectura preparada para Supabase (actualmente Mock/Local).
*   **Exportación:** Soporte nativo para Excel (`xlsx`).

## Instalación y Uso

1.  **Clonar repositorio:**
    ```bash
    git clone https://github.com/softwarelab7/Fluxo.git
    cd Fluxo
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    ```

3.  **Iniciar servidor de desarrollo:**
    ```bash
    npm run dev
    ```
